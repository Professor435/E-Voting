# E-Voting
 **SecureVote** is a modern, fully functional e-voting web application featuring a sleek glassmorphism UI with gradient animations and floating particles. It includes secure voter authentication, interactive candidate selection with real-time vote tracking, and a live results dashboard with animated progress bars. Built with vanilla JavaScript, Tailwind CSS, and Font Awesome, it simulates blockchain-style vote recording with unique transaction IDs, confetti celebrations, and responsive design—all in a single HTML file ready for immediate deployment.
